## What is Mention Python?

Mention Python is a modern Python wrapper for the Mention API that enables you to monitor and analyze brand mentions across the web. Built with modern Python features, it provides a clean, intuitive interface for accessing all Mention API endpoints.

Whether you're building social media monitoring tools, reputation management systems, or analytics dashboards, Mention Python gives you the tools to track and respond to mentions of your brand across blogs, news sites, forums, and social media platforms.

## Quick example

Here's how simple it is to get started with Mention Python:

```python
from mention import MentionClient

# Initialize the client
client = MentionClient(access_token="your-token")

# Fetch all alerts for your account
alerts = client.get_alerts(account_id="your-account-id")

# Display alert names
for alert in alerts.alerts:
    print(alert.name)
```

<Note>
The Mention API requires OAuth2 authentication. You'll need to obtain an access token from your Mention account before using this SDK.
</Note>
