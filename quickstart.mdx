This guide will walk you through making your first API calls with the Mention Python SDK. By the end, you'll be able to fetch alerts and mentions from your Mention account.

## Prerequisites

Before you begin, make sure you have:

- Python 3.14 or higher installed
- The Mention Python SDK installed (`pip install mention_python`)
- A valid Mention access token
- Your Mention account ID

<Note>
If you haven't installed the SDK yet, check out our [Installation guide](/installation).
</Note>

## Your first API call

<Steps>
  <Step title="Import and initialize the client">
    Start by importing the `MentionClient` and initializing it with your access token:

    ```python
    from mention import MentionClient

    # Initialize the client with your access token
    client = MentionClient(access_token="your-token-here")
    ```

    <Tip>
    Store your access token in environment variables rather than hardcoding it. Use `python-dotenv` to load from a `.env` file.
    </Tip>
  </Step>

  <Step title="Fetch your account information">
    Verify your connection by fetching your account details:

    ```python
    # Get your account information
    account = client.get_account_me()
    print(f"Account name: {account.name}")
    print(f"Account ID: {account.id}")
    ```

    This confirms your authentication is working and retrieves your account ID for subsequent requests.
  </Step>

  <Step title="Get all alerts">
    Fetch all monitoring alerts configured in your account:

    ```python
    # Fetch all alerts for your account
    alerts_response = client.get_alerts(account_id=account.id)

    # Display alert information
    for alert in alerts_response.alerts:
        print(f"Alert: {alert.name}")
        print(f"  ID: {alert.id}")
        print(f"  Query: {alert.query.included_keywords}")
        print()
    ```

    The response includes all alerts with their names, IDs, and query configurations.
  </Step>

  <Step title="Fetch mentions for an alert">
    Get recent mentions for a specific alert:

    ```python
    # Get the first alert
    alert = alerts_response.alerts[0]

    # Fetch mentions for this alert
    mentions = client.get_mentions(
        account_id=account.id,
        alert_id=alert.id,
        limit=10  # Fetch the 10 most recent mentions
    )

    # Display mention details
    for mention in mentions.mentions:
        print(f"Title: {mention.title}")
        print(f"Source: {mention.source.name if mention.source else 'Unknown'}")
        print(f"URL: {mention.original_url}")
        print(f"Published: {mention.published_at}")
        print()
    ```

    <Note>
    Mentions can be filtered by date, source, read status, sentiment tone, and more. See the API reference for all available filters.
    </Note>
  </Step>

  <Step title="Fetch a specific mention">
    Retrieve detailed information about a single mention:

    ```python
    # Get a specific mention by ID
    mention_id = mentions.mentions[0].id

    mention = client.get_mention(
        account_id=account.id,
        alert_id=alert.id,
        mention_id=mention_id
    )

    print(f"Title: {mention.title}")
    print(f"Description: {mention.description}")
    print(f"URL: {mention.original_url}")
    print(f"Tone: {mention.tone}")
    ```
  </Step>
</Steps>

## Complete example

Here's a complete working example that puts it all together:

```python complete_example.py
from mention import MentionClient

def main():
    # Initialize the client
    client = MentionClient(access_token="your-token")

    # Get account information
    account = client.get_account_me()
    print(f"Connected as: {account.name}\n")

    # Fetch all alerts
    alerts_response = client.get_alerts(account_id=account.id)
    print(f"Found {len(alerts_response.alerts)} alerts\n")

    # Process the first alert
    if alerts_response.alerts:
        alert = alerts_response.alerts[0]
        print(f"Alert: {alert.name}")

        # Get recent mentions
        mentions = client.get_mentions(
            account_id=account.id,
            alert_id=alert.id,
            limit=5
        )

        print(f"Recent mentions: {len(mentions.mentions)}\n")

        for mention in mentions.mentions:
            print(f"  - {mention.title}")
            print(f"    {mention.original_url}")
            print()

if __name__ == "__main__":
    main()
```

## Using async client

For better performance with concurrent requests, use the async client:

```python async_example.py
import asyncio
from mention import AsyncMentionClient

async def main():
    # Initialize async client
    async with AsyncMentionClient(access_token="your-token") as client:
        # Get account info
        account = await client.get_account_me()
        print(f"Account: {account.name}")

        # Fetch alerts
        alerts = await client.get_alerts(account_id=account.id)

        # Fetch mentions for the first alert
        if alerts.alerts:
            alert = alerts.alerts[0]
            mentions = await client.get_mentions(
                account_id=account.id,
                alert_id=alert.id,
                limit=10
            )

            for mention in mentions.mentions:
                print(f"- {mention.title}")

if __name__ == "__main__":
    asyncio.run(main())
```

<Tip>
The async client (`AsyncMentionClient`) is ideal for applications that need to make multiple API requests concurrently, such as processing mentions from multiple alerts simultaneously.
</Tip>

## Advanced filtering

Filter mentions by various criteria:

```python
from datetime import datetime, timedelta

# Fetch mentions from the last 7 days
week_ago = datetime.now() - timedelta(days=7)

mentions = client.get_mentions(
    account_id=account.id,
    alert_id=alert.id,
    limit=100,
    not_before_date=week_ago,  # Only mentions from the last week
    tone="positive",            # Only positive sentiment
    read=False,                 # Only unread mentions
    source="twitter"            # Only from Twitter
)
```

## Pagination with iter_mentions

For large result sets, use the iterator to automatically handle pagination:

```python
# Iterate over all mentions with automatic pagination
for mention in client.iter_mentions(
    account_id=account.id,
    alert_id=alert.id,
    limit=100  # Page size
):
    print(f"Processing: {mention.title}")
    # The iterator automatically fetches more pages as needed
```

## Error handling

Handle API errors gracefully:

```python
from mention import (
    MentionClient,
    MentionAuthError,
    MentionNotFoundError,
    MentionRateLimitError,
    MentionConnectionError
)

try:
    client = MentionClient(access_token="your-token")
    alerts = client.get_alerts(account_id="invalid-id")

except MentionAuthError as e:
    print(f"Authentication failed: {e}")
    print(f"Status code: {e.status_code}")

except MentionNotFoundError as e:
    print(f"Resource not found: {e}")

except MentionRateLimitError as e:
    print(f"Rate limit exceeded. Retry after: {e.retry_after} seconds")

except MentionConnectionError as e:
    print(f"Connection error: {e}")
```

<Warning>
The SDK includes automatic retry logic with exponential backoff. Rate limit errors will be retried automatically up to the configured `max_retries` value.
</Warning>

## Using context managers

Ensure proper resource cleanup with context managers:

```python
# Synchronous client
with MentionClient(access_token="your-token") as client:
    alerts = client.get_alerts(account_id="your-account-id")
    # Client is automatically closed when exiting the context

# Async client
async with AsyncMentionClient(access_token="your-token") as client:
    alerts = await client.get_alerts(account_id="your-account-id")
    # Client is automatically closed when exiting the context
```
