## Overview

Mentions are the results captured by your alerts. This guide covers how to retrieve, filter, update, and stream mentions using the Mention Python SDK.

## Getting Mentions

Retrieve mentions for a specific alert:

```python
from mention import MentionClient

client = MentionClient(access_token="your-token")

# Get mentions for an alert
response = client.get_mentions(
    account_id="your-account-id",
    alert_id="alert-123",
    limit=100
)

print(f"Found {len(response.mentions)} mentions")
for mention in response.mentions:
    print(f"- {mention.title}")
    print(f"  Source: {mention.source_name}")
    print(f"  Tone: {mention.tone}")
    print(f"  URL: {mention.original_url}")
```

### Combined Filters

Combine multiple filters for precise queries:

```python
from datetime import datetime, timedelta

# Get unread, positive Twitter mentions from the last 7 days
response = client.get_mentions(
    account_id="your-account-id",
    alert_id="alert-123",
    source="twitter",
    tone="positive",
    read=False,
    not_before_date=datetime.now() - timedelta(days=7),
    limit=100
)

print(f"Found {len(response.mentions)} matching mentions")
```

## Curating Mentions

Update mention properties like read status, favorite, tone, or tags:

<Steps>

<Step title="Import the request model">
```python
from mention import MentionClient
from mention.models import CurateMentionRequest, Tone
```
</Step>

<Step title="Create the curation request">
```python
request = CurateMentionRequest(
    read=True,
    favorite=True,
    tone=Tone.POSITIVE,
    tags=["important", "follow-up"]
)
```
</Step>

<Step title="Apply the changes">
```python
updated_mention = client.curate_mention(
    account_id="your-account-id",
    alert_id="alert-123",
    mention_id="mention-456",
    request=request
)

print(f"Updated: {updated_mention.title}")
print(f"Favorite: {updated_mention.favorite}")
print(f"Read: {updated_mention.read}")
```
</Step>

</Steps>

## Paginating Through Mentions

Manually handle pagination using cursors:

```python
cursor = None
all_mentions = []

while True:
    response = client.get_mentions(
        account_id="your-account-id",
        alert_id="alert-123",
        limit=100,
        cursor=cursor
    )
    
    all_mentions.extend(response.mentions)
    
    if not response.has_more:
        break
    
    cursor = response.links.more if response.links else None
    if not cursor:
        break

print(f"Retrieved {len(all_mentions)} total mentions")
```

## Streaming Mentions

For better handling of large result sets, use the iterator pattern:

```python
# Iterate through all mentions automatically
count = 0
for mention in client.iter_mentions(
    account_id="your-account-id",
    alert_id="alert-123",
    limit=100  # Per-page limit
):
    count += 1
    print(f"{count}. {mention.title}")
    
    # Process each mention
    if mention.tone == "negative":
        print("  ⚠️ Negative sentiment detected")

print(f"Processed {count} mentions")
```

### Streaming with Filters

Apply filters while streaming:

```python
# Stream only unread, positive mentions
for mention in client.iter_mentions(
    account_id="your-account-id",
    alert_id="alert-123",
    read=False,
    tone="positive",
    limit=50
):
    print(f"Unread positive: {mention.title}")
```

## Real-Time Mention Streaming

Stream new mentions as they arrive in real-time:

```python
from datetime import datetime

print("Starting real-time mention stream...")
print("Press Ctrl+C to stop")

try:
    for mention in client.stream_mentions(
        account_id="your-account-id",
        alert_id="alert-123",
        since=datetime.now()
    ):
        print(f"\n🔔 New mention: {mention.title}")
        print(f"   Source: {mention.source_name}")
        print(f"   URL: {mention.original_url}")
        print(f"   Published: {mention.published_at}")
except KeyboardInterrupt:
    print("\nStream stopped")
```

<Note>
The `stream_mentions()` method uses long-polling to continuously fetch new mentions as they arrive.
</Note>

## Complete Workflow Example

Here's a complete example of managing mentions:

```python
from mention import MentionClient
from mention.models import CurateMentionRequest, Tone

client = MentionClient(access_token="your-token")
account_id = "your-account-id"
alert_id = "alert-123"

# Get unread mentions
response = client.get_mentions(
    account_id=account_id,
    alert_id=alert_id,
    read=False,
    limit=50
)

print(f"Processing {len(response.mentions)} unread mentions...\n")

for mention in response.mentions:
    print(f"📄 {mention.title}")
    print(f"   Tone: {mention.tone}")
    
    # Auto-favorite positive mentions
    if mention.tone == Tone.POSITIVE:
        client.curate_mention(
            account_id=account_id,
            alert_id=alert_id,
            mention_id=mention.id,
            request=CurateMentionRequest(
                favorite=True,
                read=True
            )
        )
        print("   ⭐ Added to favorites")
    
    # Trash spam mentions
    elif mention.tone == Tone.NEGATIVE and "spam" in mention.description.lower():
        client.curate_mention(
            account_id=account_id,
            alert_id=alert_id,
            mention_id=mention.id,
            request=CurateMentionRequest(trashed=True)
        )
        print("   🗑️ Moved to trash")
    
    print()

# Mark remaining as read
client.mark_all_mentions_read(account_id, alert_id)
print("✓ All mentions marked as read")

client.close()
```

## Error Handling

Handle errors when working with mentions:

```python
from mention.exceptions import (
    MentionNotFoundError,
    MentionAPIError,
    MentionRateLimitError
)

try:
    mention = client.get_mention(account_id, alert_id, "invalid-id")
except MentionNotFoundError:
    print("Mention not found")
except MentionRateLimitError as e:
    print(f"Rate limited. Retry after {e.retry_after} seconds")
except MentionAPIError as e:
    print(f"API error: {e}")
```
