title: "Tasks"

The Task resource provides methods for creating, retrieving, updating, and deleting tasks associated with mentions.

## get_tasks

Fetch all tasks for an alert.

```python
response = client.get_tasks("account-id", "alert-id")
for task in response.tasks:
    print(f"Task: {task.description}")
```

### Parameters

<ParamField path="account_id" type="str" required>
  The account ID
</ParamField>

<ParamField path="alert_id" type="str" required>
  The alert ID
</ParamField>

### Returns

<ResponseField name="response" type="TasksResponse">
  [TasksResponse](/api/models/task-models) containing list of tasks
</ResponseField>

## create_task

Create a new task for a mention.

```python
from mention import CreateTaskRequest

request = CreateTaskRequest(
    description="Follow up with customer",
    due_date="2024-12-31T23:59:59Z",
    assigned_to="user-id"
)
task = client.create_task("account-id", "alert-id", "mention-id", request)
```

### Parameters

<ParamField path="account_id" type="str" required>
  The account ID
</ParamField>

<ParamField path="alert_id" type="str" required>
  The alert ID
</ParamField>

<ParamField path="mention_id" type="str" required>
  The mention ID
</ParamField>

<ParamField path="request" type="CreateTaskRequest" required>
  Task creation request
</ParamField>

### Returns

<ResponseField name="task" type="Task">
  Created [Task](/api/models/task-models) object
</ResponseField>

## update_task

## delete_task

Delete a task.

```python
success = client.delete_task("account-id", "alert-id", "mention-id", "task-id")
if success:
    print("Task deleted successfully")
```

<ParamField path="alert_id" type="str" required>

<ParamField path="mention_id" type="str" required>
  The mention ID
</ParamField>

<ParamField path="task_id" type="str" required>
  The task ID
</ParamField>

### Returns
