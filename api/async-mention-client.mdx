## Overview

The `AsyncMentionClient` class provides an async version of the Mention API client using Python's `async`/`await` syntax. It offers the same functionality as `MentionClient` but with non-blocking I/O operations.

## Constructor

<ParamField path="access_token" type="str" required>
  OAuth2 access token for authentication
</ParamField>

<ParamField path="base_url" type="str" default="https://api.mention.net/api">
  Base URL for the Mention API
</ParamField>

<ParamField path="timeout" type="float" default="30.0">
  Request timeout in seconds
</ParamField>

<ParamField path="max_retries" type="int" default="3">
  Maximum number of retry attempts for failed requests
</ParamField>

<ParamField path="retry_delay" type="float" default="1.0">
  Base delay between retries in seconds (uses exponential backoff)
</ParamField>

## Async Context Manager Support

The client supports async context manager protocol for automatic resource cleanup:

```python
import asyncio
from mention import AsyncMentionClient

async def main():
    async with AsyncMentionClient(access_token="token") as client:
        alerts = await client.get_alerts("account-id")
        # Client is automatically closed when exiting the context

asyncio.run(main())
```

## Resource Management

### close

Manually close the underlying HTTP client. Usually not needed if using async context manager.

```python
client = AsyncMentionClient(access_token="token")
try:
    # Use client
    await client.get_app_data()
finally:
    await client.close()
```

## Usage Examples

### Basic Async Usage

```python
import asyncio
from mention import AsyncMentionClient

async def main():
    client = AsyncMentionClient(access_token="your-access-token")
    
    try:
        # Get alerts
        alerts = await client.get_alerts("account-id")
        for alert in alerts.alerts:
            print(alert.name)
        
        # Get mentions
        mentions = await client.get_mentions("account-id", "alert-id", limit=50)
        for mention in mentions.mentions:
            print(mention.title)
    finally:
        await client.close()

asyncio.run(main())
```

### Using Async Context Manager

```python
import asyncio
from mention import AsyncMentionClient

async def main():
    async with AsyncMentionClient(access_token="token") as client:
        account = await client.get_account_me()
        alerts = await client.get_alerts(account.id)
        
        for alert in alerts.alerts:
            mentions = await client.get_mentions(account.id, alert.id, limit=10)
            print(f"{alert.name}: {len(mentions.mentions)} mentions")

asyncio.run(main())
```

### Concurrent Requests

```python
import asyncio
from mention import AsyncMentionClient

async def main():
    async with AsyncMentionClient(access_token="token") as client:
        # Fetch multiple resources concurrently
        account, app_data = await asyncio.gather(
            client.get_account_me(),
            client.get_app_data()
        )
        
        # Get mentions from multiple alerts concurrently
        alerts = await client.get_alerts(account.id)
        mention_tasks = [
            client.get_mentions(account.id, alert.id, limit=10)
            for alert in alerts.alerts[:5]
        ]
        all_mentions = await asyncio.gather(*mention_tasks)
        
        for alert, mentions in zip(alerts.alerts[:5], all_mentions):
            print(f"{alert.name}: {len(mentions.mentions)} mentions")

asyncio.run(main())
```

### Error Handling with Async

```python
import asyncio
from mention import AsyncMentionClient
from mention.exceptions import (
    MentionAuthError,
    MentionNotFoundError,
    MentionRateLimitError,
    MentionAPIError
)

async def main():
    async with AsyncMentionClient(access_token="token") as client:
        try:
            alert = await client.get_alert("account-id", "alert-id")
        except MentionAuthError:
            print("Invalid credentials")
        except MentionNotFoundError:
            print("Alert not found")
        except MentionRateLimitError as e:
            print(f"Rate limited. Retry after {e.retry_after} seconds")
            await asyncio.sleep(e.retry_after or 60)
        except MentionAPIError as e:
            print(f"API error: {e.message}")

asyncio.run(main())
```
